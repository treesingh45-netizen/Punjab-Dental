# Punjab Dental & Medical Clinic Website

Static, SEO-focused, mobile-responsive website with 5 pages:

- Home
- About
- Services
- Gallery
- Contact / Appointment

## Included features

- Online appointment form (opens WhatsApp with booking details)
- Sticky WhatsApp button
- Click-to-call links
- Google Maps embed
- Google reviews section layout
- SEO meta tags and heading structure
- Fast-loading static architecture
- Local-first image slots with fallbacks for clinic branding and media

## Image slots

Add your real media files to `assets/images` using these names:

- `punjab-logo.png`
- `dr-amir-naveed.jpg`
- `teeth-whitening-before-after.jpg`
- `clinic-interior.jpg`
- `digital-xray-equipment.jpg`
- `treatment-visual.jpg`
- `clinic-front.jpg`

## Deployment (Hosting Setup)

You can host this site on:

- cPanel shared hosting: upload all files to `public_html`
- Netlify: drag-and-drop project folder
- Vercel: import as static site
- GitHub Pages: push files and enable Pages

Main entry file is `index.html`.
